# 🌐 Modern Responsive Website (Dark & Light Mode)

A modern, clean, and fully responsive website built using **Tailwind CSS**.
This project supports both **Dark Mode 🌙** and **Light Mode ☀️**, ensuring a seamless user experience across all devices.

---

🔗 Live Demo

👉 [Live Link](https://srclogc.vercel.app/))

---

## 🚀 Features

* 🌙 Dark & Light Mode Toggle
* 📱 Fully Responsive (Mobile, Tablet, Laptop, 2XL Screens)
* ⚡ Built with Tailwind CSS
* 🎯 Pixel Perfect Design
* 💎 Modern UI / UX
* 🔄 Smooth Theme Switching (with Local Storage)
